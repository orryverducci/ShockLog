This project is maintained by orryverducci

ShockLog is developed by Orry Verducci primarily for use by Shock Radio. ShockLog supports Windows 7, Windows Server 2008 R2, Windows 8 and Windows Server 2012. .Net Framework 4.5 is required.

Building from Source

This project has been developed in Microsoft Visual Studio 2012 RC, using .Net Framework 4.5.

Before building ensure you have the required prerequisites:

  • BASS and BASSenc, placed in the x86 and x64 folders, available from http://www.un4seen.com/.
  • BASS.NET, available from http://bass.radio42.com/.
